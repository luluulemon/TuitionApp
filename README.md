# App for marking attendance

---

## Setup
Use S3 and SQL

Variables:
1. ACCESS_KEY
2. SECRET_KEY
3. SPRING_DATASOURCE_USERNAME
4. SPRING_DATASOURCE_PASSWORD
5. SPRING_MAIL_PASSWORD / HOST / PORT / USERNAME
---
